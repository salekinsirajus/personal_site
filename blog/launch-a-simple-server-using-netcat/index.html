Launch A Simple HTTP Server Using netcat

By Sirajus Salekin on July 6, 2018

netcat is a quite powerful tool, and it can be used for almost anything under the sun related to TCP/IP networking. I needed the simplest web server once to render some HTML and CSS files but did not have a way to open those files directly from a browser window. netcat came very handy, and I was surprised how powerful yet simple this tool was.

If you find yourself in the same situation, you can fire up a simple HTTP server with netcat. Here's a little guide for you:

Make a shell script with whatever commands you want to run when you make request. You potentially could run a full-blown webserver with this script, but I would not recommend it. For example, I wanted to render the result of ifconfig and so this is what the script serve.sh looks like:

#!/bin/bash

echo "Hello World!\n\n"

echo "parent directory:"
ls ..

echo "\nNetwork config\n"
echo "$(ifconfig)

Make sure to set the right execution permission:

chmod 700 serve.sh
  • Now, run the following command on terminal. This will start the server
while true; do { echo -e 'HTTP/1.0 200 OK\r\n'; sh serve.sh; } | nc -l 8080; done

Open your browser and go to localhost:8080 as you specified earlier in the command.
